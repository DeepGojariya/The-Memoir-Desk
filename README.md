# Wedding Journey Portal 🌸

A gorgeous, interactive, macOS-themed personal wedding portal. Designed with pixel-perfect details, sleek window dynamics, interactive widgets, a custom visual story journal, a dynamic polaroid scrapbook, and a collaborative digital guestbook/wishes board.

## 🛠️ Local Development

If you want to run the project locally on your computer:

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Build the production bundle:**
   ```bash
   npm run build
   ```
